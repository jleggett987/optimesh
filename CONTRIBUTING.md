# Contributing to OptiMesh

Thank you for your interest in OptiMesh.

This repository is currently being developed as a portfolio-grade distributed browser compute platform concept focused on ethical, opt-in participation and transparent system design.

## Current Contribution Status

At this early stage, the project is primarily being developed and shaped by the repository author.

External contributions may be opened up later as the architecture and implementation stabilize.

## Contribution Guidelines

For now, contributions are best limited to:

- documentation suggestions
- typo fixes
- architecture feedback
- issue reports
- questions and discussion

## Principles

Any future contributions should align with the core principles of OptiMesh:

- transparent compute participation
- explicit user opt-in
- bounded workloads
- ethical platform design
- clear architectural reasoning

## Before Contributing Code

Before larger code contributions are accepted in the future, the project should first establish:

- clearer implementation milestones
- stable component boundaries
- task execution patterns
- verification expectations

## Notes

The goal of this project is not to create hidden browser mining or opaque background compute activity.

OptiMesh is intended as a transparent, opt-in distributed compute platform concept.
